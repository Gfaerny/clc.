# clc.
Super simple timer for debian.

you can find how to report bug for clc [here](https://github.com/Gfaerny/clc./blob/main/bug_report.md)

![build](https://github.com/Gfaerny/clc./actions/workflows/build.yml/badge.svg)
# how to build

- install dependency
  - here list of dep for linux system
    -make
    -gcc
  - then use "install_dependency.sh" script
    -this script clone
      -RGFW windowing framwork
      -glyph for rendering text with OpenGL
- then run this to build
...
make
#and the use clc
./clc
...

# how to use
- keybind 
  - space : start/stop timer
  - r : restart timer
